STEP 01這個工具做什麼

填寫角色資料 → 自動生成 JSON 證據檔案 + SHA-256 時間戳 + QR Code 認證單，交給著作權局 / 文化部 / 媒體公會存查，確立你對 CHAI 虛擬角色的 IP 所有權。

完整流程
① 點「載入預設」或空白填寫
Ya-Ya / 藍龍蝦AI 預設一鍵帶入，新角色從零開始填。
② 填寫 A–H 欄位
識別 → 外觀 → 媒體 → 概念 → 功能 → 著作權 → 證據 → 認證機關。
③ 上傳角色圖片 / 影片
本地、網址、HeyGen、YouTube 四種來源，全部收錄進檔案。
④ 按「⚡ 生成」
自動產生 SHA-256 雜湊、時間戳、QR Code。
SHA-256: A3F8C2D1… · 2026-06-09 14:30 +0800
⑤ 複製 / 下載 / 列印
複製 JSON 存 GitHub、下載 .json 備份、列印 PDF 認證單交機關。

STEP 03圖片 / 影片上傳

四種來源全部收錄進 JSON，生成後在認證單上方直接顯示，作為角色外觀視覺佐證。

📁 本地上傳
點上傳區 → 手機相簿選取 → JPG / PNG / MP4。可選多個。
🔗 圖片網址
貼圖片直連（.jpg / .png 結尾）→ 點「＋」→ 即時預覽。
https://wei567.github.io/zhonghua-times/yaya.png
🎬 HeyGen
HeyGen 分享頁複製連結 → 貼入 → 嵌入播放，唇型同步影片佐證。
▶ YouTube
貼 YouTube 網址 → 自動解析嵌入。適合 Ya-Ya 播報 / Suno MV。
⚠️
本地圖片只存當下瀏覽器，關掉就不見。建議先上傳到 GitHub Pages，再用「圖片網址」方式加入，確保連結永久可驗證。
